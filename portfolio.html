Портфолио

То, что успел

Здесь я выложу свой опыт в верстке. Это не учебные проекты, это бесплатные макеты из интернета, один коммерческий проект и собственный проект сделанный в Figma с нуля. Опыт в React проектах доступен по ссылке https://epprosportz.ru/
Italy
 Простенький тест на IQ, с запросом к API и без перезагрузки страницы
Italy
 Еще один тест, наверное придется сделать отдельную вкладку, чтобы не засорять портфолио, хотя оно конечно оставляет желать лучшего
Test on BlackWall
 Просто тестовое задание, похожее на мои предыдущие пет-проекты.
Motivate
 Одна из последних работ, макет брал на просторах телеграм каналов. Адаптивная верстка с использованием "медиа-запросов", без макета для адаптива. Модальное окно при клике на кнопку.
MeatLoft
 Личный проект, в котором пришлось поработать дизайнером. На втором экране слайдера скрин из игры GTA5, для этого скрина пришлось зайти в магазин одежды, выбрать что-то похожее на одежду курьера, потом заказать велосипед в интернет-магазине и поездить по городу поискать подходящий городской пейзаж. Ну это так, немного разнообразия в работе.
Зубная Фея
 Коммерческий проект, сразу после курса верстальщика прошел стажировку в monomoloco-team, ребята доверили несрочный проект, сложность проекта была в собственной CSS библиотеке, так команда ускоряла свою работу в несколько раз, создавая макеты и верстая по собственным сеткам. Интересный опыт.
Apartments
 Данный макет нашел на просторах интернета, в тот момент когда я его верстал, в Figma видно было как еще несколько человек одновременно изучают этот макет, возможно его можно встретить в портфолио не только у меня. Данную верстку "натягивал" на WordPress, в планах квартир использовал fancybox плагин
COVID-19
 Во время карантина хотелось сверстать что-то актуальное, нашел бесплатный макет о COVID-19 сверстал, и в конце сайта подключил статистику со стороннего сервиса, немного анимировал круглые элементы на сайте.
